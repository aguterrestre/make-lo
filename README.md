Make-lo
Sistema web de facturación y administración de productos con Django 3 y PostgreSQL 12.

Este proyecto es para realizar pruebas en desarrollo y para mantener la base de datos inicial con sus tablas
de configuración correspondientes.

Make-lo es un sistema web de facturación y administración de productos que ayuda a mejorar tu negocio. Te brinda las herramientras necesarias para darle orden a tus tareas diarias y permitirte conocer información valiosa para una mejor toma de desición.

Este sistema cuenta con la posibilidad de realizar comprobantes de ventas electrónicos (Argentina) o comunes.

Esta distribuido en modulos que permiten un uso simple y práctico, pensado para una excelente experiencia de usuario.
